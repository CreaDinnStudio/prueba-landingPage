Que es una Landing Page

Una Landing Page (página de aterrizaje) es una página dentro de un sitio web, desarrollada con el único objetivo de convertir los visitantes en Leads o prospectos de ventas por medio de una oferta determinada.

Formas de pago

Los pagos se realizaran en 3 fases

Fase 1

En el primer pago se cancelara el 50% del valor total para iniciar con el proyecto

Fase 2

En el segundo pago se cacela un 25% del pago total, junto a este pago se ensearan los avances que se llevan hasta el momento

Fase 3

Una vez terminado el proyecto se procedera a cancelar el ultimo 25% del pago para llegar al pago total, posteriormente se hace entrega del proyecto al cliente.
